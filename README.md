# Fake E-Commerce

Building Fake E-Commerce With Flutter and [fakestoreapi](https://fakestoreapi.com/docs)

## Dependency
- flutter_svg: ^2.0.7
- email_validator: ^2.1.17
- flutter_otp_text_field: ^1.1.1
- flutter_bloc: ^8.1.3
- http: ^1.1.0
- equatable: ^2.0.5
- card_swiper: ^3.0.1 To animate product card on main screen's header
